# HTE / MAE Framework Ledger

## FR-HTE-TD-001 — Transdutation Boundary Layer

**Status:** rooted → branching  
**Return phrase:** Find the mode that crosses.  
**Core invariant:** Transdutation is state reorganization across boundary-conditioned resonance without claiming nuclear identity change.  
**HTE role:** Adds a middle category between transduction and transmutation.  
**MAE role:** Tests whether the state change is measurable, repeatable, and boundary-dependent.

### Claim ladder

- **Transduction:** a carrier changes form.
- **Transdutation:** a state reorganizes under geometry, phase, carrier, and boundary constraint.
- **Transmutation:** nuclear identity changes and therefore requires nuclear or isotope evidence.

## FR-HTE-SK-001 — Spirograph-to-Kagome Lattice Engine

**Status:** sprout → rooted  
**Return phrase:** Phase finds the gate; lattice tests the hold.  
**Core invariant:** A spirograph phase path can be projected onto kagome boundary geometry to identify candidate coupling nodes, recurrence loops, defects, void routes, and mode-locking regions.

## FR-HTE-POA-001 — Periodic Orbit Atlas

**Status:** rooted candidate  
**Return phrase:** Close the full state.  
**Core invariant:** A true periodic orbit must close the full state, not merely draw a beautiful trace.

### POA fields

- Orbit ID
- Orbit source
- Period estimate
- Closure error
- Loop count
- Flip count
- Winding number
- Symmetry class
- Stability score
- Perturbation test
- Orbit class
- HTE harmonic tag
- Design translation

## Pipeline

> Spirograph draws candidate recurrence.  
> Periodic Orbit Atlas classifies the orbit.  
> Kagome tests whether the orbit can hold as boundary geometry.  
> MAE decides whether the held orbit causes measurable state reorganization.

## Quarantine rule

Visual overlap is not proof. Beauty is not evidence. Measurement decides.

Nothing reaches a transmutation investigation by geometry alone.
