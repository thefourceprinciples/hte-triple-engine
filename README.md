# HTE / MAE Triple Engine

Public working build for the Harmonic Table of Elements / Materials Atlas Engine project.

## Current online module

**Transdutation POA Cockpit**

This build integrates four layers:

1. **HTE identity layer** — choose an element or material focus.
2. **Spirograph phase key** — generate recurrence paths from phase parameters.
3. **Periodic Orbit Atlas** — classify orbit closure, loop behavior, winding, stability, and perturbation response.
4. **Kagome boundary scaffold** — project the orbit onto a constraint lattice and route candidate activation nodes.
5. **MAE gate classifier** — separate visual design, carrier response, boundary dependence, repeatable state reorganization, material memory, and hard-evidence investigation.

## Core doctrine

> Transduction converts a carrier.  
> Transdutation reorganizes a state.  
> Transmutation changes nuclear identity.

This project is a design and measurement cockpit. It does not claim elemental transmutation from geometry alone. Any nuclear or isotope claim requires external evidence.
